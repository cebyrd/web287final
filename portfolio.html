Portfolio

The Stokes County website was created in my Mobile Web Design class. We focused heavily on Bootstrap and responsive design. The task for this assignment was to re-do the older version of the Stokes County NC website. We used Boostrap and altered the color scheme, images, logo and layout of information.

The Planted app was created for my Mobile Interface Design class. This class we focused on create apps for mobile and wearable devices. Our goal was to create a new app idea that could provide some real life function to users. I created the Planted app which would help keep your plants alive. The app would track things such as water, sunlight and health of the plant. You could create a schedule for watering and fertilizing on the calendar. The app was made to include banner ads but could offer it ad-free for a subscription cost.

This portfolio page was one of the first I created for my Web Design class. We focused on mainly using HTML and CSS but worked with Bootstrap towards the end. This website showcases some of my original pieces that I, unfortunately, no longer have access to.
